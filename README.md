# Design-Patterns-TicTacToe-Game

TicTacToe Game, Includes all the following Design Patterns:
- Memento
- Decorator
- Adapter
- Singleton 
- Command

Technologies:
- Java Programming language 
- Eclipse IDE
- JavaFX
